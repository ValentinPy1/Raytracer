Detailed Description

Interface for lights. In the config file, in order to create a light object, you need to specify the type of the light (by specifying which base plugin to use) and the parameters of the light. The exact parameters may depend on which plugin is used.

A light does not need to inherit from the Plugin class, but it may avoid a segfault if somebody tries to load it as a plugin.
