Detailed Description

template<typename ArgType, typename... Args>
class pp::Pipeline< ArgType, Args >

Utility class to automatize calling a series of functions. Note: Most of the time, the user is better of using a simple std::vector<std::function<...>>. Use it if you need to change the order of the functions at runtime or if you need to add functions at runtime.
