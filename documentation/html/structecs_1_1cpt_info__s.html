Detailed Description

The cptInfo_s struct contains the information about a component. It contains a handle to the dynamic library, a pointer to the create function which returns a pointer to the component, a pointer to the destroy function which takes a pointer to the component as a parameter in order to destroy it, and the typeId of the component.

Upon creation, the structure will load the dynamic library, and get the create and destroy functions. Upon destruction, the structure will close the dynamic library.
